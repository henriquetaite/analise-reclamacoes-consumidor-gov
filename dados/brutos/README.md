# Dados Brutos

Esta pasta é destinada aos arquivos brutos originais utilizados no projeto.

Os arquivos brutos não foram incluídos neste repositório devido ao tamanho dos arquivos.

## Fonte dos dados

Os dados foram obtidos no portal de dados abertos do Ministério da Justiça e Segurança Pública, na base pública de reclamações do Consumidor.gov.br.

Link da base:  
https://dados.mj.gov.br/dataset/reclamacoes-do-consumidor-gov-br

## Período utilizado

Ano de referência: 2025.

## Como reproduzir

Para reproduzir o projeto, baixe os arquivos originais na fonte oficial e armazene-os localmente nesta pasta, mantendo a estrutura:

```text
dados/brutos/
├── 2025-01.xlsx
├── 2025-02.xlsx
├── 2025-03.xlsx
├── ...
└── 2025-12.xlsx
