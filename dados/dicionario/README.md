# Dicionário de Dados

Esta pasta armazena o dicionário de dados oficial da base pública do Consumidor.gov.br.

O dicionário foi utilizado para entender:

- o significado das colunas da base;
- os tipos de dados esperados;
- as regras de cálculo dos indicadores;
- os campos relacionados a resolução, satisfação, resposta e situação da reclamação;
- as limitações e cuidados necessários na análise.
